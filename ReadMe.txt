实验要求见 doc 目录下的实验指南。
pdf 版本是为了照顾 markdown 查看不方便的同学，由于转换上有一些问题，所以可能不太美观，另外在显示一些字符的时候会出现一些问题，因此建议有条件的同学直接看 markdown 文件。助教是直接在 CLion 上写的 markdown，因此在该 IDE 下得到的效果最佳，顺便该 IDE 在你们写 C/C++ 的时候也能帮助你们减少不少麻烦。（没有收广告费的安利）

call.h 里包含了你们需要实现的接口。

bottom.h 里包含了底层的模拟接口。

type.h 里包含了一些对类型名字的修改，主要是为了好看一点。

bottom.o 里包含了 bottom.h 里的接口的实现。

test.c 里包含了一些基本的测试。你们可以调用来检查自己的代码的完成情况。具体每个测试是干什么的请看实验指南。
